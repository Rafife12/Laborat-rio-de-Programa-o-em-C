## 🛠️ Como usar

- 📥 Clone ou baixe o repositório.
- 💻 Compile os arquivos C com `gcc nome_arquivo.c -o nome_arquivo` e execute.
- 💻 Compile os arquivos C++ com `g++ nome_arquivo.cpp -o nome_arquivo` e execute.
